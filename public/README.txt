Place your logo image files here:

- logo.png — used for the site header and favicon (`/logo.png`)
- favicon.ico — optional Windows favicon

Notes:
- Drop the pasted image into this folder with the filename `logo.png`.
- Vite serves files from `public/` at the site root, so the app will use `/logo.png` automatically.
- If you want a transparent background, replace `logo.png` with a transparent PNG or SVG named `logo.png`.
